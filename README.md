# 📘 概要
このリポジトリは、現場業務で利用するWebベースのツール群をまとめています。  
スマートフォンやPCブラウザから直接利用可能で、インストール不要です。  
GitHub Pagesを通じて公開され、黒板作成・原価管理・画像編集・印刷作業を効率化します。

## 📱 公開ツール一覧

- [現場原価管理台帳システム v25](https://hy11-hiro.github.io/hy11-html-tools/genba_cost_v25.html)  
  月別原価集計・工番別検索・拠点別ランキングなどができる原価管理ツール。

  **主な機能**
  - **月次原価集計**：指定期間（月単位）での原価合計・内訳（材料費・労務費・外注費など）を集計。  
  - **工番別検索・フィルタ**：工番、現場名、担当者、拠点、ステータスなどで絞り込み可能。  
  - **拠点別ランキング**：拠点ごとの原価合計や粗利率でランキング表示。上位/下位の比較が容易。  
  - **期間比較**：前月比・前年同月比の差分表示と増減率の算出。  
  - **CSV/Excel 出力**：表示中の一覧をそのまま CSV/XLSX でエクスポート。ヘッダ順は画面表示に準拠。  
  - **データインポート**：現場原価データを CSV から取り込み、既存データと突合・更新可能。  
  - **簡易グラフ表示**：月次推移や拠点別構成を棒グラフ／折れ線で可視化（画面内プレビュー）。  
  - **サマリ行・ドリルダウン**：一覧のサマリ（合計・平均）表示と、各工番から詳細画面へ遷移可能。  
  - **テンプレート出力**：会計システムや他ツール向けの出力フォーマットをプリセットで選択可能。  
  - **入力検証**：インポート時の必須項目チェックと簡易エラーレポート出力。  
  - **権限表示制御**：表示項目や出力可否はユーザ権限に応じて制御（サーバ側での権限チェック前提）。
  - **統合ダッシュボード**：Chart.js を採用し、月次推移や拠点別利益率を棒グラフ・円グラフで視覚的に把握可能。
  - **ローカルデータベース（Dexie.js）**：ブラウザ内にデータを安全に保存。ページを閉じてもデータが保持され、大量のデータも高速に検索・フィルタリング。
  - **高度なエクスポート**：ExcelJS を活用し、書式を維持したまま XLSX 形式での出力に対応。
  - **リアルタイム分析**：利益率 10% 未満の警告表示や、担当者別・拠点別のランキングを自動集計。
  - **マスタ連携とインポート**：CSV/Excel からのデータ取り込みに対応し、既存の会計データとの突合もスムーズ。

  **利用シーン**
  - 拠点・担当者ごとの採算管理と差異分析
  - 決算前後の原価進捗確認と利益予測
  - 経営判断のためのグラフ資料作成
  - 月次決算前の原価チェック・差異分析  
  - 拠点別のコスト管理と改善ポイントの抽出  
  - 工番単位での原価推移確認と担当者へのフィードバック  
  - 他システムへのデータ連携（CSV/XLSX）


- [スマホ向け黒板アプリ](https://hy11-hiro.github.io/hy11-html-tools/blackboard_mobile_v1.html)  
  現場を選ばず使える、スマホ最適化済みの汎用黒板編集ツール。シンプルながらも、詳細な文字調整や一括処理に対応しています。

  **主な機能**
  - **表示ラベル切り替え**：「工種」と「工事場所」をワンタッチで切り替え可能。現場のルールに柔軟に対応。
  - **個別サイズ調整**：工事名・工種・備考・日付のフォントサイズを個別にスライダーで微調整。
  - **直感的な操作**：ドラッグ＆ドロップによる黒板配置、サイズ（大・中・小）のクイック変更。
  - **バッチ処理（一括編集）**：複数枚の写真を同時に読み込み、リストで切り替えながら連続して黒板を合成・保存。
  - **高い視認性**：現場写真に最適な丸ゴシック系フォントと、操作しやすい Material Icons を採用。

  **利用シーン**
  - 外構工事・造成現場での日常的な写真整理
  - 状況に合わせて「工事場所」を表示したい現場での撮影
  - 大量の現場写真をまとめて処理する必要がある事務作業

- [OHD専用 黒板エディタ](https://hy11-hiro.github.io/hy11-html-tools/OHD_Blackboard_Editor.html)  
  OHD現場の撮影基準に特化した高度な黒板合成ツール。文字ごとのスケール調整や台数管理モードを搭載。

  **主な機能**
  - **項目別フォント調整**：工事名・工種・備考・日付ごとに文字サイズを個別に微調整可能。
  - **台数入力モード**：車両台数などのカウント表記をスムーズに行える専用入力欄。
  - **レイアウト自由自在**：ドラッグ＆ドロップでの配置移動、行間設定、黒板サイズ（大・中・小）の即時切替。
  - **一括管理（バッチ機能）**：複数枚の写真を読み込み、リストで切り替えながら効率的に編集・保存。
  - **視認性重視のデザイン**：現場写真でも文字が潰れにくい「M PLUS Rounded 1c」等のフォントを採用。

  **利用シーン**
  - OHD基準での工事写真撮影および整理
  - 複雑な備考内容を読みやすく配置したい場合
  - 車両搬入や資材搬入の記録作成

- [図面積算ツール Exterior Planner Pro v2](https://hy11-hiro.github.io/hy11-html-tools/drawing_estimator.html)  
  CADライクな操作感を実現した、プロフェッショナル向けの外構積算・計測ツール。PDF図面や画像から、距離・面積・数量を迅速に算出します。

  **主な機能**
  - **プロ仕様のCAD UI**：長時間の作業でも疲れにくいダークモードUIと、直感的なツールパネルを採用。
  - **PDF/画像ダイレクト読み込み**：高精度なPDFレンダリングエンジンを搭載し、図面ファイルをそのまま読み込んで計測可能。
  - **高度な計測モード**：連続線による距離計測、多角形による面積算出、クリックによる個数カウントに対応。
  - **部材マスタ・見積連携**：Excel/CSVから単価マスタを取り込み、計測結果に基づいた合計金額をリアルタイムに集計。
  - **マルチエクスポート**：計測結果を描き込んだ図面をPNG/PDFで保存できるほか、集計データをCSVで出力可能。
  - **レイヤー管理**：計測した要素をリストで管理し、個別に削除や属性変更が可能。

  **利用シーン**
  - 図面PDFからの迅速な外構面積・土量算出
  - 現場写真や図面への寸法・数量の描き込み
  - 積算データと単価マスタを組み合わせた概算見積の作成

- [計算ツール Exterior Estimator](https://hy11-hiro.github.io/hy11-html-tools/exterior-estimator.html)  
  画像やPDFを読み込み、縮尺設定後に距離・面積・体積・材料数量を算出する実務向け計算ツール。部材マスタ（Excel/CSV）連携で概算金額を出力でき、結果はCSV/PDF/PNGで保存可能。現場での迅速な積算・見積作業に最適です。 
  - **縮尺設定**：2点指定でピクセル→実寸変換  
  - **距離計測**：連続線で長さ算出  
  - **面積計測**：多角形でm²算出  
  - **個数カウント**：クリックで本数集計  
  - **体積・材料計算**：掘削量・コンクリート量・砕石量など自動算出  
  - **部材マスタ連携**：Excel/CSVから単価を読み込み概算金額を算出  
  - **出力**：CSV / PDF / PNG（計測図）

- [SMILE マスター編集ツール v25 (SMILE Master Edit Tool)](https://hy11-hiro.github.io/hy11-html-tools/smile_master_edit_tool.html)  
  SMILE V Air 連携用データの結合・編集・変換を支援する統合ツール。Excel と SMILE 出力テキストを結合して編集用データを作成します。UTF-16LE 対応、金額自動反映、端数処理などに対応。  
  - **入力**：校別データ（Excel）、現場マスター（テキスト/CSV）  
  - **主な機能**：拠点フィルタ、金額自動反映、端数処理（四捨五入等）、編集用データ出力（CSV/Excel）  
  - **主な操作**：ファイルのドラッグ＆ドロップ、拠点コードでの絞り込み、金額自動反映のON/OFF切替、変換結果のプレビュー  
  - **出力形式**：UTF-16LE 形式のテキスト、CSV、Excel（XLSX）をサポート。SMILE 取り込み要件に合わせた整形済み出力が可能。  
  - **利用シーン**：SMILE 出力データの前処理、現場データ結合、原価データの一括編集や検証作業の効率化。  
  - **注意事項**：大容量ファイルはブラウザメモリに依存するためステージングでの動作確認を推奨。SMILE のレイアウト変更がある場合は入力ファイルのヘッダ構成を確認してください。  
  - **開発メモ**：クライアントサイド完結の実装。ファイル名は `smile_master_edit_tool.html` を採用。スクリーンショットやサンプルデータを `assets` フォルダに置くと README 表示が分かりやすくなります。  
  - **更新履歴**：v25 リリースで UTF-16LE 対応と金額自動反映ロジックを強化。今後の予定はエラーログ出力と大容量処理の最適化。

- [売上データ照合・集計 (Sales Data Checker)](https://hy11-hiro.github.io/hy11-html-tools/sales-data-checker.html)  
  現場マスタと売上データを突き合わせ、重複チェックや乖離の特定、自動集計を行う実務ツール。

  - **重複チェック**：同一データの二重計上をハッシュ値で高精度に検知

  - **Excel出力**：チェック結果と集計データをそのまま XLSX 形式でエクスポート
  - **利用シーン**：月次決算時の売上計上漏れ確認、マスタ登録不備の監査作業

  - **複数ファイル結合**：複数の請求データを 1 つの一覧に統合  
  - **検証機能**：必須項目の欠落チェック、金額整合性チェック  
  - **Excel/CSV 出力**：変換後データをそのまま XLSX / CSV 形式でエクスポート  
  - **利用シーン**：支店ごとに異なる請求書の標準化、会計システム取り込み前の整形、月次締め処理の効率化
