# 📘 概要
このリポジトリは、現場業務で利用するWebベースのツール群をまとめています。  
スマートフォンやPCブラウザから直接利用可能で、インストール不要です。  
GitHub Pagesを通じて公開され、黒板作成・原価管理・画像編集・印刷作業を効率化します。
---

## 📱 公開ツール一覧

- [黒板編集 ver.3](https://hy11-hiro.github.io/hy11-html-tools/blackboard_editor_v3.html)  
  画像に黒板を合成するツール。工事名・工種・日付などを自由に配置可能。

- [現場原価管理台帳システム v25](https://hy11-hiro.github.io/hy11-html-tools/genba_cost_v25.html)  
  月別原価集計・工番別検索・拠点別ランキングなどができる原価管理ツール。

  **主な機能**
  - **月次原価集計**：指定期間（月単位）での原価合計・内訳（材料費・労務費・外注費など）を集計。  
  - **工番別検索・フィルタ**：工番、現場名、担当者、拠点、ステータスなどで絞り込み可能。  
  - **拠点別ランキング**：拠点ごとの原価合計や粗利率でランキング表示。上位/下位の比較が容易。  
  - **期間比較**：前月比・前年同月比の差分表示と増減率の算出。  
  - **CSV/Excel 出力**：表示中の一覧をそのまま CSV/XLSX でエクスポート。ヘッダ順は画面表示に準拠。  
  - **データインポート**：現場原価データを CSV から取り込み、既存データと突合・更新可能。  
  - **簡易グラフ表示**：月次推移や拠点別構成を棒グラフ／折れ線で可視化（画面内プレビュー）。  
  - **サマリ行・ドリルダウン**：一覧のサマリ（合計・平均）表示と、各工番から詳細画面へ遷移可能。  
  - **テンプレート出力**：会計システムや他ツール向けの出力フォーマットをプリセットで選択可能。  
  - **入力検証**：インポート時の必須項目チェックと簡易エラーレポート出力。  
  - **権限表示制御**：表示項目や出力可否はユーザ権限に応じて制御（サーバ側での権限チェック前提）。

  **利用シーン**
  - 月次決算前の原価チェック・差異分析  
  - 拠点別のコスト管理と改善ポイントの抽出  
  - 工番単位での原価推移確認と担当者へのフィードバック  
  - 他システムへのデータ連携（CSV/XLSX）


- [スマホ向け黒板アプリ](https://hy11-hiro.github.io/hy11-html-tools/blackboard_mobile_v1.html)  
  スマホ対応の黒板編集ツール。カメラ撮影・文字入力・画像保存まで可能。

- [LiveCameraBoard](https://hy11-hiro.github.io/hy11-html-tools/LiveCameraBoard.html)  
  カメラ映像に黒板テンプレートを重ねて撮影できる試験導入版。

- [粗利率マスター編集](https://hy11-hiro.github.io/hy11-html-tools/gross_margin_master.html)  
  Excelデータ結合・抽出ツール NextGen。原価集計データと現場マスターを結合し、フィルタリングやExcel出力が可能。

- [画像印刷 (Print Layout Builder)](https://hy11-hiro.github.io/hy11-html-tools/print_layout_builder.html)  
  画像を複数枚まとめて印刷レイアウトに配置するツール。  
  - 1ページあたりの枚数を指定可能（1〜32枚）  
  - 並び順プリセット（行優先／列優先／ジグザグ／逆順／ドラッグ順）  
  - 用紙サイズ（A4/A3/Letter）と縦横選択  
  - サムネイルクリックで拡大プレビュー  
  - ページ番号印刷オプションあり  

- [ファイル名入力 (画像ファイル名修正ツール)](https://hy11-hiro.github.io/hy11-html-tools/filename_editor_full.html)  
  画像ファイルにラベルを付けて一括ダウンロードできるツール。  
  - 画像を選択するとカード表示され、ファイル名を入力可能  
  - ラベルボタンで素早く名前を付与  
  - Shift/Ctrlキーで複数選択、Enterキーで次の画像へ移動  
  - サイズ表示（小／中／大）切替  
  - 一括ダウンロード機能あり  

- [図面積算ツール (Exterior Planner)](https://hy11-hiro.github.io/hy11-html-tools/drawing_estimator.html)  
  図面から面積や数量を計算する外構積算ツール。PDFや画像を読み込み、スケール設定後に線・多角形・カウントで計測できます。  
  - **距離計測**：連続線で長さを算出  
  - **面積計測**：多角形で範囲を指定し、m²単位で算出  
  - **個数カウント**：基数や本数をクリックで集計  
  - **部材マスタ管理**：単価表をExcel/CSVから取り込み、部材ごとの単価・色を設定可能  
  - **出力機能**：PNG画像保存、PDF出力、CSVエクスポート対応  
  - **積算リスト**：計測結果を一覧化し、合計概算金額を自動計算

- [図面積算ツール ver.2 (Exterior Planner)](https://hy11-hiro.github.io/hy11-html-tools/drawing_estimator.ver2.html)  
一時停止
  - **改善点**：edge版と統合　データ表示改善


- [計算ツール Exterior Estimator](https://hy11-hiro.github.io/hy11-html-tools/exterior-estimator.html)  
  画像やPDFを読み込み、縮尺設定後に距離・面積・体積・材料数量を算出する実務向け計算ツール。部材マスタ（Excel/CSV）連携で概算金額を出力でき、結果はCSV/PDF/PNGで保存可能。現場での迅速な積算・見積作業に最適です。 
  - **縮尺設定**：2点指定でピクセル→実寸変換  
  - **距離計測**：連続線で長さ算出  
  - **面積計測**：多角形でm²算出  
  - **個数カウント**：クリックで本数集計  
  - **体積・材料計算**：掘削量・コンクリート量・砕石量など自動算出  
  - **部材マスタ連携**：Excel/CSVから単価を読み込み概算金額を算出  
  - **出力**：CSV / PDF / PNG（計測図）

## 🧪 新機能・試験導入ツール

- [作図練習ツール (Drafting Canvas)](https://hy11-hiro.github.io/hy11-html-tools/DraftingCanvasApp.html)  
  図面作成の基礎練習や寸法表記の練習ができる作図支援ツール。  
  - **グリッド切替**：直交／アイソメ／一点透視／二点透視に対応  
  - **寸法記入**：線・矩形・円に寸法を自動表示  
  - **スナップ機能**：グリッドや交点に吸着して正確な描画が可能  
  - **PDF出力**：作図結果をそのままPDF保存可能  
  - **Reactベース**：軽量かつ高速動作、スマホでも快適に利用可能  
  - **用途例**：新人研修・図面練習・寸法感覚の習得・設計補助など
 
- [売上入力ツール](https://hy11-hiro.github.io/hy11-html-tools/sales_input_tool.html)  
  工事原価・売上を入力・管理するWeb版ツール。Excelコピペ対応で、現場マスタや得意先マスタと連携可能。  
  - **基本情報入力**：取引区分・得意先・担当者・商品名を入力  
  - **明細入力**：Excelからコピー＆ペーストで一括入力対応  
  - **マスタ管理**：現場・得意先・商品・担当者をファイル取込／保存  
  - **テンプレート機能**：よく使う入力パターンを保存・選択可能  
  - **出力機能**：テキストファイル出力、プレビュー表示、CSV編集対応  
  - **検証機能**：入力値の自動チェックでエラー防止  
  
- [仕入れ入力ツール](https://hy11-hiro.github.io/hy11-html-tools/purchase_input_tool.html)  
  工事原価・仕入れを入力・管理する Web ツール。Excel のコピー＆ペーストに対応し、仕入先マスタや商品マスタと連携できます。  
  - **基本情報入力**：取引区分・仕入先・担当者・商品名を入力  
  - **明細入力**：Excel からの一括貼付けに対応  
  - **マスタ管理**：仕入先・商品・担当者をファイル取込／保存  
  - **テンプレート機能**：よく使う入力パターンを保存・選択可能  
  - **出力機能**：テキスト出力・プレビュー・CSV 編集対応  
  - **検証機能**：入力値の自動チェックでエラーを防止

- [SMILE マスター編集ツール v25 (SMILE Master Edit Tool)](https://hy11-hiro.github.io/hy11-html-tools/smile_master_edit_tool.html)  
  SMILE V Air 連携用データの結合・編集・変換を支援する統合ツール。Excel と SMILE 出力テキストを結合して編集用データを作成します。UTF-16LE 対応、金額自動反映、端数処理などに対応。  
  - **入力**：校別データ（Excel）、現場マスター（テキスト/CSV）  
  - **主な機能**：拠点フィルタ、金額自動反映、端数処理（四捨五入等）、編集用データ出力（CSV/Excel）  
  - **主な操作**：ファイルのドラッグ＆ドロップ、拠点コードでの絞り込み、金額自動反映のON/OFF切替、変換結果のプレビュー  
  - **出力形式**：UTF-16LE 形式のテキスト、CSV、Excel（XLSX）をサポート。SMILE 取り込み要件に合わせた整形済み出力が可能。  
  - **利用シーン**：SMILE 出力データの前処理、現場データ結合、原価データの一括編集や検証作業の効率化。  
  - **注意事項**：大容量ファイルはブラウザメモリに依存するためステージングでの動作確認を推奨。SMILE のレイアウト変更がある場合は入力ファイルのヘッダ構成を確認してください。  
  - **開発メモ**：クライアントサイド完結の実装。ファイル名は `smile_master_edit_tool.html` を採用。スクリーンショットやサンプルデータを `assets` フォルダに置くと README 表示が分かりやすくなります。  
  - **更新履歴**：v25 リリースで UTF-16LE 対応と金額自動反映ロジックを強化。今後の予定はエラーログ出力と大容量処理の最適化。

- [工事写真台帳メーカー (Construction Photo Ledger Maker)](https://hy11-hiro.github.io/hy11-html-tools/photo_ledger_maker.html)  
  工事現場の写真を整理し、台帳形式で管理・出力できるツール。Excelテンプレートを取り込んで罫線や結合セルを保持したまま出力可能。テンプレート無しでも標準レイアウトで台帳を生成できます。  
  - **写真管理**：現場写真をアップロードし、台帳形式でプレビュー  
  - **テンプレート対応**：Excelテンプレートを読み込み、罫線・結合セルを保持  
  - **テンプレート無し出力**：標準レイアウトでExcel台帳を生成（左：写真／右：情報）  
  - **表紙生成**：標準・表形式・モダンの3パターンを自動生成し差し替え可能  
  - **出力形式**：Excel (XLSX) 出力に対応、Meiryo UIフォント適用  
  - **利用シーン**：工事写真の提出資料作成、現場管理、報告書作成の効率化  
  - **注意事項**：大容量写真の取り込みはブラウザメモリに依存するため、ステージング環境での確認を推奨

- [売上データ照合・集計 (Sales Data Checker)](https://hy11-hiro.github.io/hy11-html-tools/sales-data-checker.html)  
  現場マスタと売上データを突き合わせ、重複チェックや乖離の特定、自動集計を行う実務ツール。
  - **データ突合**：現場マスタと売上CSVを読み込み、工番・金額・得意先などの不一致を自動抽出
  - **重複チェック**：同一データの二重計上をハッシュ値で高精度に検知
  - **自動集計**：得意先別・拠点別の売上合計をリアルタイムに算出
  - **Excel出力**：チェック結果と集計データをそのまま XLSX 形式でエクスポート
  - **利用シーン**：月次決算時の売上計上漏れ確認、マスタ登録不備の監査作業
 
- [AI-OCR Invoice Processor.html](https://hy11-hiro.github.io/hy11-html-tools/AI-OCR%20Invoice%20Processor.html)  
  請求書画像・PDFをAIが解析し、品名・数量・金額などを自動抽出する軽量AI-OCRツール。  
  **主な機能**：AI解析、レイアウト非依存抽出、CSV出力、スマホ対応、HTML単体で動作  
  **利用シーン**：個人の請求書整理、AI-OCRの検証、Excel集計前の前処理  
  **重要**：解析には Google Gemini API を使用するため、画像データは外部サーバーへ送信されます。  
  **個人利用専用ツールです。会社の請求書・機密情報には使用しないでください。**

## 🛠 開発者向け情報

- 各ツールはHTML/JSベースで構成されており、GitHub Pages上で直接ホストされています。  
- ReactやTailwind CSS、IndexedDB、jsPDFなどのライブラリを活用し、軽量かつ拡張性の高い設計を採用。  
- ツールごとに独立した構成のため、機能追加やUI改修が容易です。  
- Pull RequestやIssueによる改善提案も歓迎しています。
