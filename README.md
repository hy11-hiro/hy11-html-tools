# 📘 概要
このリポジトリは、現場業務で利用するWebベースのツール群をまとめています。  
スマートフォンやPCブラウザから直接利用可能で、インストール不要です。  
GitHub Pagesを通じて公開され、黒板作成・原価管理・画像編集・印刷作業を効率化します。
---

## 📱 公開ツール一覧

- [黒板編集 ver.3](https://hy11-hiro.github.io/hy11-html-tools/blackboard_editor_v3.html)  
  画像に黒板を合成するツール。工事名・工種・日付などを自由に配置可能。

- [現場原価管理台帳システム v25](https://hy11-hiro.github.io/hy11-html-tools/genba_cost_v25.html)  
  月別原価集計・工番別検索・拠点別ランキングなどができる原価管理ツール。

- [スマホ向け黒板アプリ](https://hy11-hiro.github.io/hy11-html-tools/blackboard_mobile_v1.html)  
  スマホ対応の黒板編集ツール。カメラ撮影・文字入力・画像保存まで可能。

- [LiveCameraBoard](https://hy11-hiro.github.io/hy11-html-tools/LiveCameraBoard.html)  
  カメラ映像に黒板テンプレートを重ねて撮影できる試験導入版。

- [粗利率マスター編集](https://hy11-hiro.github.io/hy11-html-tools/gross_margin_master.html)  
  Excelデータ結合・抽出ツール NextGen。原価集計データと現場マスターを結合し、フィルタリングやExcel出力が可能。

- [画像印刷 (Print Layout Builder)](https://hy11-hiro.github.io/hy11-html-tools/print_layout_builder.html)  
  画像を複数枚まとめて印刷レイアウトに配置するツール。  
  - 1ページあたりの枚数を指定可能（1〜32枚）  
  - 並び順プリセット（行優先／列優先／ジグザグ／逆順／ドラッグ順）  
  - 用紙サイズ（A4/A3/Letter）と縦横選択  
  - サムネイルクリックで拡大プレビュー  
  - ページ番号印刷オプションあり  

- [ファイル名入力 (画像ファイル名修正ツール)](https://hy11-hiro.github.io/hy11-html-tools/filename_editor_full.html)  
  画像ファイルにラベルを付けて一括ダウンロードできるツール。  
  - 画像を選択するとカード表示され、ファイル名を入力可能  
  - ラベルボタンで素早く名前を付与  
  - Shift/Ctrlキーで複数選択、Enterキーで次の画像へ移動  
  - サイズ表示（小／中／大）切替  
  - 一括ダウンロード機能あり  

- [図面積算ツール (Exterior Planner)](https://hy11-hiro.github.io/hy11-html-tools/drawing_estimator.html)  
  図面から面積や数量を計算する外構積算ツール。PDFや画像を読み込み、スケール設定後に線・多角形・カウントで計測できます。  
  - **距離計測**：連続線で長さを算出  
  - **面積計測**：多角形で範囲を指定し、m²単位で算出  
  - **個数カウント**：基数や本数をクリックで集計  
  - **部材マスタ管理**：単価表をExcel/CSVから取り込み、部材ごとの単価・色を設定可能  
  - **出力機能**：PNG画像保存、PDF出力、CSVエクスポート対応  
  - **積算リスト**：計測結果を一覧化し、合計概算金額を自動計算

- [図面積算ツール ver.2 (Exterior Planner)](https://hy11-hiro.github.io/hy11-html-tools/drawing_estimator.ver2.html)  
一時停止
  - **改善点**：edge版と統合　データ表示改善


- [計算ツール Exterior Estimator](https://hy11-hiro.github.io/hy11-html-tools/exterior-estimator.html)  
  画像やPDFを読み込み、縮尺設定後に距離・面積・体積・材料数量を算出する実務向け計算ツール。部材マスタ（Excel/CSV）連携で概算金額を出力でき、結果はCSV/PDF/PNGで保存可能。現場での迅速な積算・見積作業に最適です。 
  - **縮尺設定**：2点指定でピクセル→実寸変換  
  - **距離計測**：連続線で長さ算出  
  - **面積計測**：多角形でm²算出  
  - **個数カウント**：クリックで本数集計  
  - **体積・材料計算**：掘削量・コンクリート量・砕石量など自動算出  
  - **部材マスタ連携**：Excel/CSVから単価を読み込み概算金額を算出  
  - **出力**：CSV / PDF / PNG（計測図）

## 🧪 新機能・試験導入ツール

- [作図練習ツール (Drafting Canvas)](https://hy11-hiro.github.io/hy11-html-tools/DraftingCanvasApp.html)  
  図面作成の基礎練習や寸法表記の練習ができる作図支援ツール。  
  - **グリッド切替**：直交／アイソメ／一点透視／二点透視に対応  
  - **寸法記入**：線・矩形・円に寸法を自動表示  
  - **スナップ機能**：グリッドや交点に吸着して正確な描画が可能  
  - **PDF出力**：作図結果をそのままPDF保存可能  
  - **Reactベース**：軽量かつ高速動作、スマホでも快適に利用可能  
  - **用途例**：新人研修・図面練習・寸法感覚の習得・設計補助など
 
- [売上入力ツール](https://hy11-hiro.github.io/hy11-html-tools/sales_input_tool.html)  
  工事原価・売上を入力・管理するWeb版ツール。Excelコピペ対応で、現場マスタや得意先マスタと連携可能。  
  - **基本情報入力**：取引区分・得意先・担当者・商品名を入力  
  - **明細入力**：Excelからコピー＆ペーストで一括入力対応  
  - **マスタ管理**：現場・得意先・商品・担当者をファイル取込／保存  
  - **テンプレート機能**：よく使う入力パターンを保存・選択可能  
  - **出力機能**：テキストファイル出力、プレビュー表示、CSV編集対応  
  - **検証機能**：入力値の自動チェックでエラー防止  
  
- [仕入れ入力ツール](https://hy11-hiro.github.io/hy11-html-tools/purchase_input_tool.html)  
  工事原価・仕入れを入力・管理するWeb版ツール。Excelコピペ対応で、仕入先マスタや商品マスタと連携可能。  
  - **基本情報入力**：取引区分・仕入先・担当者・商品名を入力  
  - **明細入力**：Excelからコピー＆ペーストで一括入力対応  
  - **マスタ管理**：仕入先・商品・担当者をファイル取込／保存  
  - **テンプレート機能**：よく使う入力パターンを保存・選択可能  
  - **出力機能**：テキストファイル出力、プレビュー表示、CSV編集対応  
  - **検証機能**：入力値の自動チェックでエラー防止

## 🛠 開発者向け情報

- 各ツールはHTML/JSベースで構成されており、GitHub Pages上で直接ホストされています。  
- ReactやTailwind CSS、IndexedDB、jsPDFなどのライブラリを活用し、軽量かつ拡張性の高い設計を採用。  
- ツールごとに独立した構成のため、機能追加やUI改修が容易です。  
- Pull RequestやIssueによる改善提案も歓迎しています。
